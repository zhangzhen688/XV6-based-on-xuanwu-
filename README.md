# XV6-based-on-xuanwu-

本仓库为基于 xuanwu 版本的 XV6 教学/实验系统的镜像与源码导入。包含内核源码、用户态程序、构建脚本和测试用例。

1. 安装 RISC-V 工具链（示例）

```bash
# 示例：Ubuntu 下安装 riscv 工具链（请根据实际需要调整）
```

项目（拟）

- **项目名称：** 面向玄铁C910架构的轻量级多任务操作系统研发（基于 RISC-V RV64GC）
- **技术栈：** RISC-V 汇编、C/C++、内核架构设计、驱动开发、MMU/内存管理

主要目标要点：

- RISC-V 内核架构设计与硬件适配：针对玄铁（XuanTie）C906/C910 处理器特性，从引导、SBI 调用到内核初始化实现完整启动流程；实现 Sv39 多级页表与 MMU 映射，建立内核/用户态内存隔离。
- 核心子系统与并发控制：实现抢占式调度、多进程管理、Trap 处理与上下文切换；实现自旋锁与睡眠锁以支持多核并发；引入 CoW 写时复制以优化 fork 内存开销。
- 外设驱动与日志文件系统：实现基于 PLIC/CLINT 的中断框架，UART 与磁盘驱动；设计支持日志（journaling）的崩溃一致性文件系统，保证元数据事务性。
